## 1. 목표

개발자 도움 없이 콘텐츠, 상품, 카테고리, 주문, 판매자, 검색, 배너를 운영할 수 있는 관리자 시스템을 구축한다.

## 5. 화려한 사용자 화면과의 분리

관리자 화면은 화려함보다 빠른 업무와 정확성을 우선한다.

- 모션 최소화
- 데이터 밀도 높임
- 키보드 조작
- 대량 작업
- 검색과 필터
- 변경 이력
- 오류 복구

## 6. 산출물

- `/admin`
- 역할 기반 권한
- 콘텐츠 편집기
- 상품 편집기
- 주문 관리
- 사용자 관리
- 검색 분석
- 감사 로그
- 파일 라이브러리

## 7. 완료 조건

- 모든 중요 변경이 기록된다.
- 승인 없이 의료정보가 공개되지 않도록 설정할 수 있다.
- 상품 유형별 필드가 동적으로 바뀐다.
- 대량 수정과 내보내기 기능의 정책이 있다.
- 관리자 API가 일반 사용자 API와 권한상 분리된다.
