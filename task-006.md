## 1. 목표

키워드 검색, 카테고리 탐색, 자연어 질문을 통합한 플랫폼 핵심 검색 경험을 구축한다.

## 2. 1차 검색 기능

- 자동완성
- 최근 검색어
- 인기 검색어
- 오타 보정
- 동의어
- 초성 또는 띄어쓰기 보정
- 콘텐츠·상품 통합 결과
- 유형별 탭
- 다중 필터
- 정렬
- 결과 내 재검색
- 검색 없음 추천

## 3. AI 검색 원칙

AI는 진단하지 않는다.

AI 결과는 다음 구조로 제한한다.

- 질문 요약
- 관련 일반정보
- 관련 증상 또는 주제
- 응급 신호
- 관련 콘텐츠
- 생활관리 정보
- 관련 상품
- 의료전문가 상담 안내
- 근거 또는 출처 링크

## 4. 검색 데이터

- 질환명
- 증상명
- 일반어
- 전문용어
- 약어
- 동의어
- 오타
- 연령·성별·상황
- 상품명
- 브랜드
- 상품 특성

## 5. 산출물

- `/search`
- `/ask`
- 검색 API
- 자동완성 API
- 검색 로그
- 검색 결과 없음 로그
- 동의어 사전
- 검색 분석 대시보드 기초
- AI 응답 안전 가드레일 문서

## 6. 완료 조건

- 키워드 검색과 자연어 질문이 구분된다.
- 검색 결과가 정보와 상품을 명확히 구분한다.
- 응급 신호가 있을 때 판매 CTA보다 먼저 안내한다.
- AI 응답에는 의료적 한계가 표시된다.
- 검색 실패 키워드를 관리자가 확인할 수 있다.
