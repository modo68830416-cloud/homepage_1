# 브랜드 원칙

- 작업 ID: TASK-001
- 관련 문서: [[project-vision]], [[visual-benchmark-matrix]]

## 1. 화려함과 신뢰성의 균형 원칙

화려함은 장식이 아니라 "브랜드 기억 · 콘텐츠 이해 · 상품 탐색"을 강화하는 도구로만 허용한다. 아래 우선순위를 모든 화면 설계의 상위 규칙으로 삼는다.

1. **의료적 정확성과 안전** — 어떤 비주얼 결정도 이 원칙을 훼손할 수 없다.
2. **가독성과 이해도** — 정보 영역은 항상 화려함보다 우선한다.
3. **탐색 효율** — 사용자가 원하는 정보/상품에 도달하는 속도.
4. **브랜드 임팩트** — 위 세 가지를 해치지 않는 선에서 최대치로 표현한다.

## 2. 디자인 원칙 (판단 기준, 10개 이상)

1. **효과는 예산제다.** 페이지 유형별 시각효과 예산(TASK-003 §5)을 넘지 않는다.
2. **본문 근처에는 화려함을 두지 않는다.** 건강정보 본문 컨테이너 내부에는 파티클/3D/시차 효과를 직접 넣지 않는다.
3. **CTA 위계는 응급 > 신뢰정보 > 콘텐츠 탐색 > 상품 추천 순이다.**
4. **모든 화려한 요소는 실패해도 안전하다(Graceful Degradation).** 3D/모션이 로드되지 않아도 핵심 기능은 100% 동작한다.
5. **판매 유형은 배지로 정직하게 표시한다.** 제휴/위탁/입점 여부를 숨기지 않는다.
6. **타이포그래피는 두 얼굴을 가진다.** 브랜드 영역은 대담하게, 정보 영역은 중립적으로.
7. **모션은 의미를 전달할 때만 사용한다.** 장식용 반복 애니메이션은 지양한다.
8. **다크 패턴을 만들지 않는다.** 구독 해지, 주문 취소를 어렵게 만드는 UI를 금지한다.
9. **일관된 토큰만 사용한다.** 임의의 하드코딩된 색상/간격 값을 금지한다(TASK-003 토큰 체계).
10. **모바일이 먼저다.** 데스크톱 전용 화려함(예: 커서 반응형 3D)은 모바일 대체안을 항상 동반한다.
11. **접근성은 화려함의 예외가 아니라 전제조건이다.** `prefers-reduced-motion`, 키보드 탐색, 명도 대비를 만족하지 못하는 효과는 출시하지 않는다.
12. **성능 예산을 넘는 화려함은 화려함이 아니라 결함이다.** LCP/INP/CLS 기준(TASK-009)을 넘는 연출은 자동 축소한다.

## 3. 브랜드 톤 & 보이스

- 정보 영역: 전문적이되 권위적이지 않음, 공감형 문장, 과장 부사(놀라운, 기적의 등) 금지
- 브랜드/캠페인 영역: 대담하고 감각적인 문장 허용, 그러나 의학적 주장은 이 영역에서도 금지
- 커머스 영역: 명확한 사실 기반 설명, 비교 가능한 스펙 제공
