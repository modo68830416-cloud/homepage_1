# 프로젝트 비전

- 작업 ID: TASK-001
- 문서 상태: 초안 (전략 확정 단계)

## 1. 비전 선언문

> 사용자가 건강과 관련해 궁금한 정보를 신뢰할 수 있는 방식으로 찾고, 필요한 상품과 서비스를 한 곳에서 연결하는 몰입형 건강 라이프스타일 플랫폼

이 한 문장이 이후 모든 UI, 콘텐츠, 상품, 기술 판단의 최상위 기준이다. 어떤 결정이든 이 문장으로 되돌아가 검증한다.

## 2. 비전 해석

| 구절 | 의미 | 판단 기준으로 사용하는 방법 |
|---|---|---|
| 신뢰할 수 있는 방식으로 찾고 | 출처·검수자·수정일이 항상 보여야 함 | 신뢰 요소가 빠진 콘텐츠 화면은 출시 불가 |
| 필요한 상품과 서비스를 한 곳에서 연결 | 정보와 커머스가 별개 사이트처럼 분리되지 않음 | 콘텐츠 상세에는 관련 상품 슬롯이 항상 존재 |
| 몰입형 | 2026 비주얼 키워드(임srb 3D, 시네마틱 모션 등)를 실제로 사용 | 홈·랜딩·상품상세는 효과 예산 65 이상 |
| 라이프스타일 플랫폼 | 질병 치료가 아니라 일상적 건강 관리가 중심 | 의료행위·진단으로 오인되는 표현 금지 (TASK-006, TASK-009 참조) |

## 3. 5년 지향점 (비코드 방향성)

- 검색 한 번으로 "이해 → 판단 → 구매/예약"까지 이어지는 경험을 표준으로 만든다.
- 자체판매에 국한되지 않고 제휴·위탁·입점·디지털·구독·예약을 모두 동일한 품질 기준으로 운영한다.
- 화려한 비주얼이 브랜드 자산이 되되, 의료 신뢰도를 해치는 순간 화려함보다 신뢰를 우선한다.

## 4. 비전과 상충하는 것으로 간주하는 패턴

- 과장 광고형 카피로 건강정보를 왜곡하는 것
- 상품 CTA가 응급 안내보다 시각적으로 우선하는 레이아웃
- 화려함을 이유로 본문 가독성을 희생하는 것
- 판매 유형에 따라 사용자에게 불리하게 정보가 숨겨지는 것 (예: 제휴 표시 누락)
