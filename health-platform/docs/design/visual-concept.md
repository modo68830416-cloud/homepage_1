# 비주얼 콘셉트

- 작업 ID: TASK-003
- 상위 원칙: [[brand-principles]] (효과 예산제, 영역별 이원화)

## 1. 세 개의 시각 영역

이 플랫폼은 하나의 통일된 "스타일"이 아니라 세 개의 영역별 시각 언어를 명시적으로 분리해 사용한다. 세 영역은 같은 토큰 체계([[design-tokens]])를 공유하되 적용 강도가 다르다.

### 브랜드 영역 (Hero, 캠페인, 랜딩)

- 대형 가변 타이포그래피, 딥 그라데이션, 유기적 3D 오브젝트
- 빛/입자/유리(Glassmorphism)/반투명 레이어
- 비대칭 에디토리얼 그리드, 강한 스크롤 리듬
- 효과 예산: 100 (TASK-003 §5 기준)

### 정보 영역 (콘텐츠 본문, 검사 결과 등)

- 밝고 안정적인 배경, 높은 가독성, 넓은 행간(1.7~1.85)
- 명확한 정보 계층(H1~H4, 캡션, 데이터 스타일 분리)
- 출처·수정일 강조 컴포넌트 상시 노출
- 시선 분산을 줄이기 위해 모션은 fade/slide 정도로 제한
- 효과 예산: 30 (본문 기준)

### 커머스 영역 (상품 목록/상세)

- 고해상도 상품 이미지, 촉각적 질감(그레인/소프트 섀도우)
- 카드보다 "무대형" 프레젠테이션 우선 (상품 상세)
- 비교/구매 CTA는 항상 고정 위치에서 명확하게 노출
- 판매 유형 배지 상시 노출
- 효과 예산: 65(목록) / 80(상세)

| 페이지 | 영역 | 효과 예산 |

| 관리자 | 실무 | 0~10 |

## 3. 반복 모티프 (브랜드 기억 요소)

- **유기적 블롭(organic blob) 그라데이션**: Hero 배경 및 섹션 전환부에서 반복 사용해 브랜드 시그니처로 만든다.
- **떠 있는 3D 오브젝트**: 신체 부위를 추상화한 유기적 형태(사실적 인체 렌더링은 지양 — 의료 오인 방지).
- **유리 레이어 카드**: 브랜드 영역의 통계/신뢰 지표 카드에 반투명 유리 질감 적용, 정보 영역에는 사용하지 않음(가독성 우선).

## 4. 사용하지 않는 것

- 실사 인체/장기 3D 렌더링(의료 정확성 오인 위험)
- 정보 본문 내 자동재생 배경 영상
- 결제/장바구니 화면의 파티클·블러 효과
