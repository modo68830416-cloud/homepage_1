## 3. 이 저장소 구조에 대한 중요한 안내

이 플랫폼(`health-platform/`)은 기존 저장소의 MERION/MOMO 홈페이지와 **완전히 독립된 디렉터리**로 만들어졌다(작업 시작 시 사용자 선택). 그러나 현재 **하나의 Git 저장소(`codespaces-nextjs`) 안의 하위 폴더**일 뿐, 별도의 Git 저장소로 분리되지는 않았다.

`.github/workflows/ci.yml`/`deploy.yml`은 관례상 **Git 저장소 루트**에 있어야 GitHub Actions가 인식한다. 이 폴더를 별도 GitHub 저장소로 분리(`git init` 후 새 원격 저장소에 push)하기 전까지는 이 워크플로 파일들이 실제로 실행되지 않는다. → [결정 필요] 이 프로젝트를 별도 저장소로 분리할지, 아니면 워크플로를 상위 저장소 루트로 옮기고 `working-directory: health-platform`만 유지할지.
